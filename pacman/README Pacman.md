# Pacman

Pacman is a simple coding exercise I completed as part of the MIT xPro full-stack developer program. This is a very simple version of Pacman that only goes from side-to-side horizontally on one line.

## Usage

Most of the code in Pacman was provided by the instructors, while I had to add code to run the function at a certain interval and provide boundaries at the screen width that would cause the Pacman image to reverse course.
